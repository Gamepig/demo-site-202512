Design Features

結合最新設計趨勢，打造現代化、響應式的 Web 體驗

Bento Grid

靈活的 12 欄網格系統，支援 6 種區塊尺寸，完美適應各種內容需求。

Glassmorphism

磨砂玻璃效果配合 Aurora 極光漸層，創造層次豐富的視覺體驗。

View Transitions

原生頁面轉場動畫 API，無需框架即可實現流暢的頁面切換效果。

Fluid Typography

使用 CSS clamp() 實現流體排版，文字大小隨視窗自動縮放，無需媒體查詢即可完美適應所有裝置。

Responsive Design

4 種響應式斷點完美適配桌面、平板與手機，Bento Grid 自動調整欄位數量與區塊尺寸。
